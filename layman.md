CANZAP is a command-line tool that reads recordings of vehicle network traffic — the electrical messages cars and trucks constantly send between their computers — and checks them against a set of rules you define. Think of it like a spell-checker, but for the internal chatter of a vehicle: you tell it "the engine RPM signal must appear at least three times" or "there should never be a fault code on this channel," and it tells you pass or fail. It is aimed at security researchers, automotive engineers, and QA teams who want to catch problems in firmware or hardware changes before they ship, and who want those checks to run automatically in a build pipeline just like a software test suite.
